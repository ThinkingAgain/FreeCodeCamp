使用h标签显示内容的层次关系

标签本身都应语义化，不应仅仅为了获得不同字号而使用不同级别的标题标签。视觉样式，但我们应该使用 CSS 来调整。

相同级别（或者更高级别）的标题标签用于开启新的章节，低一级别的标题标签用于开启上一级标题标签的子小节。

HTML5新标签

main标签

标签用于呈现网页的主体内容，且每个页面只能有一个。这意味着它只应包含与页面中心主题相关的信息，而不应包含如导航连接、网页横幅等可以在多个页面中重复出现的内容

section - 有联系的内容组; article - 独立完整的内容

如果一本书是一个article的话，那么每个章节就是section。当内容组之间没有联系时，可以使用div。

@media (max-width: 100px) { /* CSS Rules */ }当设备宽度小于或等于 100px 时

为优化图片在高分辨率设备下的显示效果，最简单的方式是定义它们的 width 和 height 值为源文件宽高的一半。

视窗单位

vw：如 10vw 的意思是视窗宽度的 10%。

vh： 如 3vh 的意思是视窗高度的 3%。

vmin： 如 70vmin 的意思是视窗中较小尺寸的 70% (高度 VS 宽度)。

vmax： 如 100vmax 的意思是视窗中较大尺寸的 100% (高度 VS 宽度)。
